## 🎯 Overview

This project is designed to provide a practical introduction to database systems, covering core concepts such as:

- **Database Design**: Entity-Relationship (ER) modeling and normalization
- **SQL Fundamentals**: CRUD operations, queries, and data manipulation
- **Database Management**: Creating, maintaining, and optimizing databases
- **Data Integrity**: Keys, constraints, and relationships

## 🚀 Project Goals

- Understand relational database architecture and design principles
- Build hands-on experience with SQL and database operations
- Develop skills in data modeling and schema design
- Learn best practices for database optimization and maintenance
- Collaborate as a team on a real-world database project

## ✨ Features

- Comprehensive database schema with proper normalization
- Sample data for testing and demonstrations
- Query examples and documentation
- Best practices and coding standards
- Educational materials and learning resources

## 🛠️ Getting Started

### Prerequisites

- Database management system (MySQL, PostgreSQL, SQLite, etc.)
- Database client or IDE (MySQL Workbench, pgAdmin, DBeaver, etc.)
- Git for version control

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/bshaw333/introduction_to_database.git
   cd introduction_to_database
   ```

2. Set up the database:
   - Create a new database in your DBMS
   - Execute the schema files located in the `schema/` directory
   - Load sample data if provided

3. Connect your database client to the created database
